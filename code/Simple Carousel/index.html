Options

Property Type Default Description
vertical boolean false
size integer 3 The number of visible elements.
scroll integer 1 The number of items to scroll by.
speed integer 500 The speed of scroll animation.
prev string '.carousel-prev' Ex: '#prev'
next string '.carousel-next' Ex: '#next'
